# 🏙️ Tarsus Belediyesi Akıllı Şehir Sistemi - Frontend

Bu proje, Tarsus Belediyesi Akıllı Şehir ve Şikayet Yönetim Sistemi'nin **Frontend (Kullanıcı Arayüzü)** kısmıdır. Vatandaşların şikayet bildirebildiği ve yetkililerin bu şikayetleri harita üzerinden yönetebildiği modern bir React uygulamasıdır.

This is the Frontend repository for the Tarsus Municipality Smart City System, built with React.js. It provides dynamic interfaces for Citizens, Auditors, and Admins.

## 🚀 Teknolojiler (Tech Stack)
* **Kütüphane:** React.js
* **Routing:** React Router Dom
* **HTTP İstemcisi:** Axios
* **Bildirimler:** SweetAlert2
* **Harita Entegrasyonu:** Leaflet & React-Leaflet *(Planlanan/Eklenen)*
* **Stil:** Saf CSS (Modern & Responsive Design)

## 🎯 Özellikler (Features)
* **Akıllı Navigasyon (Smart Navbar):** Kullanıcı rolüne göre değişen dinamik menü.
* **Rol Bazlı Yönlendirme (Role-Based Routing):** Admin, Denetmen (Auditor) ve Vatandaş (Citizen) için özel sayfalar.
* **Kullanıcı Yönetim Paneli:** Adminlerin yetki verebileceği dinamik tablo arayüzü.
* **Harita Görünümü:** Şikayetlerin lokasyon bazlı ve renk kodlu (Yeni, İşlemde, Çözüldü) gösterimi.
